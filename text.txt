filtering 
you can filter on the frontend, but for your app, you really shouldn’t rely on it long-term.
Real-world apps don’t do this

Apps like:

GitHub
LinkedIn
Airbnb

👉 ALWAYS filter on the backend

Because:

databases are optimized for filtering
frontend is not

For YOUR app specifically

You are building:

developer marketplace
search + filters
profiles
projects

👉 This is 100% backend-filtered territory

⚖️ Simple rule to remember

If data comes from an API → filter on the API
If data is static/local → filter in React
